## Steps to install this project

##### 1. Clone the repository
```
$ git clone https://github.com/alexcastillla/Contact-List-API-REACT-FLUX
```
##### 2. Install the /node_modules
```
$ npm install
```
##### 3. Run the webpack development server
```
$ npm run start
```
